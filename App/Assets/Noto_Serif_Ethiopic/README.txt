Noto Serif Ethiopic Variable Font
=================================

This download contains Noto Serif Ethiopic as both a variable font and static fonts.

Noto Serif Ethiopic is a variable font with these axes:
  wdth
  wght

This means all the styles are contained in a single file:
  NotoSerifEthiopic-VariableFont_wdth,wght.ttf

-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
